Bayonet 是应卓开发的 Spring Boot 4.x 增强库，提供可复用的自动配置、工具类和集成支持，涵盖
Web、安全、数据（JPA/Redis/MongoDB）、校验、日志等领域。

### 关于 Claude Code

本项目使用 Claude Code 作为AI辅助开发Agent。请参考[CLAUDE.md](./CLAUDE.md)。<br>
大语言模型使用的是 deepseek-v4-flash / deepseek-v4-pro
